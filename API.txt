Fizz-Buzz REST Server API Documentation
Base URL
The base URL for all endpoints is http://localhost:5000 (replace with your actual server address).

1. Generate Fizz-Buzz Sequence
Endpoint: GET /fizzbuzz
Generate a Fizz-Buzz sequence based on user-defined parameters.

Parameters:

int1 (optional, default: 3): Integer - Multiples of this number replaced with str1.
int2 (optional, default: 5): Integer - Multiples of this number replaced with str2.
limit (optional, default: 15): Integer - Upper limit for generating the Fizz-Buzz sequence.
str1 (optional, default: "fizz"): String - String replacement for multiples of int1.
str2 (optional, default: "buzz"): String - String replacement for multiples of int2.
Example:

bash
Copy code
GET http://localhost:5000/fizzbuzz?int1=3&int2=5&limit=15&str1=fizz&str2=buzz
Response:

json
Copy code
[
    "1", "2", "fizz", "4", "buzz", "fizz", "7", "8", "fizz", "buzz", "11", "fizz", "13", "14", "fizzbuzz"
]
2. Statistical Endpoint
Endpoint: GET /fizzbuzz/stats
Retrieve information about the most used Fizz-Buzz request.

Parameters:

None
Example:

bash
Copy code
GET http://localhost:5000/fizzbuzz/stats
Response:

json
Copy code
{
    "int1": 3,
    "int2": 5,
    "limit": 15,
    "str1": "fizz",
    "str2": "buzz",
    "hits": 25
}
Error Handling
For any invalid or missing parameters, the server will return an appropriate error response with details.
Status Codes
200 OK: Successful request.
400 Bad Request: Invalid request parameters.
500 Internal Server Error: Server-side error.
